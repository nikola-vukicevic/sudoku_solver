# sudoku_solver

Sudoku solver koji traži rešenje korak po korak (WIP; za sada još uvek ima bagova).

Za sada implementirane tehnike:

- Prosta eliminacija kandidata ukrštanjem reda, kolone i bloka
- jedini kandidat
- skriveni singl
- upereni par
- skriveni par

U planu:

- tripleti
- X-wing
- XY-wing

(Verovatno i sve ostale tehnike.)
